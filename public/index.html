  2. XML DOM define um padrão para acessar e manipular documentos XML. O DOM apresenta um documento XML como uma árvore. Todo documento é um nodo da árvore. Cada elemento XML é um nodo de elemento. O texto nos elementos são nodos de texto. Cada atributo é um nodo de atributo.
